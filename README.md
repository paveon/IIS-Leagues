# [BUT FIT 2018] Leagues
IIS (Information Systems) university course project.
Django based information system for management of
e-sports related data.

## Startup
1) Run `python3 manage.py migrate` in root folder to migrate tables
2) Start Django debug HTTP server with `python manage.py runserver`
3) Navigate to `http://127.0.0.1:8000` in browser

## Dependencies (Ubuntu)
* **[Python3.6+](https://www.python.org/)**
    * **[Django 3.1+](https://www.djangoproject.com/)**
* **[jQuery 3.3.1](https://jquery.com/)**
